# Prerequisites
__Frontend:__
Make sure you have `npm` installed

```bash
cd frontend
npm install
```
__Backend:__
Make sure you have python3 installed (including pip)
```bash
cd backend
pip install -r requirements.txt
```

# Running
__Backend:__
```bash
cd frontend
python3 server.py
```

__Frontend:__
```bash
cd frontend
npm run dev
```


# Code
Interesting __Frontend__ code is in `frontend/src/App.tsx`, interesting __Backend__ in `backend/server.py`.
